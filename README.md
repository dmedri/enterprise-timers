enterprise-timers
=================

This is a running set of systemd .timers for  periodic tasks (see. onCalendar= settings).

OnCalendar syntax accept these options:

- usec, us (microsecond)
- msec, ms (millisecond)
- seconds, second, sec, s
- minutes, minute, min, m
- hours, hour, hr, h
- days, day, d
- weeks, week, w
- months, month, M
- years, year, y
